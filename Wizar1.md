My final recommendation

Don't think of this repository as "another document archive."

Think of it as the operating system for your educational work. Your BEJ articles, masterclasses, learning reports, and future experiments all become applications running on the same architecture. That separation between the core framework and the publications themselves should make the project easier to maintain and extend over time.
